

## 特性
- 二次封装实用高扩展性组件
- 响应式、多主题，多配置，快速集成，开箱即用
- 最新技术栈，使用 `Vue3`、`Typescript`、`Pinia`、`Vite` 等前端前沿技术
- 强大的鉴权系统，对路由、菜单、功能点等支持`三种鉴权模式`，满足不同的业务鉴权需求
- 持续更新，实用性页面模板功能和交互，随意搭配组合，让构建页面变得简单化

## 使用

- 获取项目代码

```bash
git clone https://gitlab.com/hnkdpt/hp-staffPos-manange.git
```

- 安装依赖

```bash
cd naive-ui-admin

yarn install

```

- 运行

```bash
yarn dev
```

- 打包

```bash
yarn build
```
